Περιγραφή
Η εργασία υλοποιεί ένα απλό σύστημα διαχείρισης κρατήσεων ξενοδοχείου σε Java, με χρήση αντικειμενοστραφούς προγραμματισμού, UML διαγραμμάτων και προαιρετικού SQL/ERD μοντέλου βάσης δεδομένων.

Το σύστημα υποστηρίζει:

Προβολή διαθέσιμων δωματίων

Αναζήτηση δωματίων ανά τύπο

Δημιουργία κράτησης με έλεγχο διαθεσιμότητας

Ακύρωση κράτησης

Προβολή κρατήσεων πελάτη ή δωματίου

Υπολογισμό κόστους κράτησης

Αναζήτηση κρατήσεων ανά χρονικό διάστημα
